meteor-plivo
===============

Meteor wrapper for Plivo NodeJs helper library.

##Install
```bach
meteor add risul:plivo
```

##Use
Add this to your settings.json or METEOR_SETTINGS env variable containing all settings as JSON

```json
{
    "plivo": {
        "authId": "********",
        "authToken": "********"
}
```
